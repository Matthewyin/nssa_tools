MLG 深色主题配色测试

测试方案五：经典深色配色方案 - Tailwind CSS Slate色系

配色方案

  • 画布背景：#1E293B (Slate-800)
  • 容器背景：#334155 (Slate-700)
  • 顶层立方体：#FFFFFF (纯白色)
  • 下层立方体：蓝灰色系渐变

视觉优势

  • • 高对比度：白色立方体极其醒目
  • • 护眼舒适：深色背景减少眼疲劳
  • • 现代感强：符合当前UI设计趋势
  • • 层次分明：立方体层级清晰可辨
  • • 专业感：类似专业软件界面

深色主题特点：

  • 画布：#1E293B - 深邃的石板色，提供稳定的视觉基础
  • 容器：#334155 - 稍浅的石板色，形成层次感
  • 边框：白色半透明，在深色背景上清晰可见
  • 立方体：白色顶层如明珠般闪耀，蓝灰下层层次丰富
  • 整体效果：高级、专业、现代、护眼
